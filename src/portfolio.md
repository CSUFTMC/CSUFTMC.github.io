## 🎮 服务器列表概览

以下是当前开放的 CSUFTMC 社团服务器信息。请根据你需要的版本和玩法选择对应的服务器。

---

### 🌲 CSUFTMC 主服务器 (常驻/多版本)

这是一个支持 **Java 和基岩版全部版本**的常驻主服务器，提供长期支持。

| 特性 | 详情 |
| :--- | :--- |
| **支持版本** | 全部版本 (Java/基岩版) |
| **游戏模式** | 多世界生存，无限拓展性 |
| **登录模式** | 支持离线登录 |
| **描述** | CSUFTMC 主服务器，支持多世界，无限拓展性，支持离线登录，提供长期支持。 |

#### 连接地址 (CSUFTMC 主服务器)

| 描述 | 地址 |
| :--- | :--- |
| 移动ipv6直连 (推荐) | `ipv6.csuftmc.dpdns.org:25565` |
| 广东三网(通用|备用) | `a.csuftmc.dpdns.org:55907` |
| 广东电信(速度快|电信专用) | `hns.csuftmc.dpdns.org:25574` |

---

### ⛏️ 生存服4 (Java 1.21.10 - 限时)

这是一个小型的生存日常服。请注意，它**不提供长期支持**，生命周期结束后会提供存档文件。

| 特性 | 详情 |
| :--- | :--- |
| **支持版本** | **Java 1.21.10** |
| **游戏模式** | 小生存日常服 |
| **长期支持** | 否 (限时开放，结束后提供存档) |
| **描述** | 小生存日常服。不提供长期支持，生命周期结束会提供存档文件。 |

#### 连接地址 (生存服4)

| 描述 | 地址 |
| :--- | :--- |
| 移动ipv6直连 (推荐) | `ipv6.csuftmc.dpdns.org:25566` |
| 广东三网(通用|备用) | `a.csuftmc.dpdns.org:37384` |

---

### 🦴 脆骨症3 (Java 1.19.2 - 模组服)

这是一款围绕胸腔模组展开的**偏冒险向主题整合包**。

| 特性 | 详情 |
| :--- | :--- |
| **支持版本** | **Java 1.19.2** |
| **游戏类型** | 主题冒险/模组服 (需客户端) |
| **描述** | 一款围绕胸腔展开的偏冒险向的主题整合包... 允许玩家顺畅、充分的体验所有胸腔的相关内容以及其延展内容。 |

| :--- | :--- |
