# ⌨️ 玩家指令百科

在 CSUFTMC 的世界中，熟练使用指令能让您的生存之旅事半功倍。

::: tip 💡 指令输入小技巧
* **自动补全**：输入指令前几个字母后按 `Tab` 键，可自动补全玩家 ID 或指令单词。
* **历史记录**：按键盘上的 `↑` 或 `↓` 方向键，可以快速切换之前输入过的指令。
* **参数说明**：`< >` 代表必填参数，`[ ]` 代表选填参数。输入时**请勿包含**这些括号。
:::

---

## 🔐 账号与安全 (AuthMe)
**服务器采用离线+AuthMe验证模式，保护账号不被冒充是首要任务。**

| 指令 | 功能说明 | 备注 |
| :--- | :--- | :--- |
| `/register <密码> <密码>` | **注册**账号 | 仅限首次进入服务器时使用 |
| `/login <密码>` | **登录**账号 | 再次进入服务器时使用 |
| `/cp <旧密码> <新密码>` | 修改账号密码 | 建议定期更换以确保安全 |
| `/authme logout` | 登出当前账号 | 在公共电脑游玩时建议下线前执行 |

## 🏡 领地插件 (Residence)
**保护您的建筑、容器和牲畜不受侵犯。**

::: tip 详细操作指引
关于如何使用木棍选区、设置领地进出消息等进阶操作，请参阅主条目：
👉 **[领地插件详细教程](./res_help.md)**
:::

| 指令 | 功能说明 | 备注 |
| :--- | :--- | :--- |
| `/res select` | 开启/关闭**选区模式** | 使用木棒左键点 A 点，右键点 B 点 |
| `/res create <领地名>` | **创建**选定区域为领地 | 需支付游戏金币 |
| `/res pset <玩家> <权限> t/f` | 给指定玩家**领地权限** | 见下方的常用权限参数表 |
| `/res tpa <领地名>` | 传送到指定的领地 | 快速回到自己的基地 |
| `/res set` | 打开领地**可视化设置菜单** | 图形化开关领地权限 |
| `/res remove <领地名>` | 撤销并删除领地 | 确认删除需输入两次 |

::: details 🔑 常用领地权限 (Flags) 参数表
在执行 `/res pset` 时，您可以替换以下权限名：
* `use`: 使用门、拉杆、压力板等。
* `container`: 开启箱子、熔炉、漏斗等容器。
* `build`: 放置或破坏方块。
* `tp`: 是否允许他人传送到您的领地。
* `animallist`: 交互/击杀领地内的动物。
:::

## 💰 经济与贸易 (QuickShop / GS)
**通过箱子商店 或全球市场 与全服玩家进行贸易。**
::: tip 详细操作指引
关于如何详细创建、管理商店等进阶操作，请参阅主条目：
👉 **[商店教程](./shop_help.md)**
:::
| 指令 | 功能说明 | 备注 |
| :--- | :--- | :--- |
| `/qs buy` | 切换商店为**收购模式** | 对着自己的箱子商店输入 |
| `/qs price <单价>` | 修改当前商店的售价 | 对着商店输入 |
| `/gs` | 打开**全球虚拟市场** | 随时随地跨世界买卖 |
| `/gs sell <总价>` | 上架手持物品到全球市场 | 记得去“暂存箱”提取卖出的钱 |
| `/money` | 查看您的当前余额 | 掌握您的财富状况 |
| `/pay <玩家ID> <金额>` | 向指定玩家**转账** | 面对面交易或结算工钱时使用 |

## 🚀 传送与社交
**解决迷路、快速回家或寻找伙伴。**

| 指令 | 功能说明 | 备注 |
| :--- | :--- | :--- |
| `/spawn` | 瞬间回到**服务器出生点** | 遇到紧急情况的逃生手段 |
| `/sethome [名称]` | 在当前位置**设置家** | 默认最多设置 5 个家 |
| `/home [名称]` | 传送回您设置的家 | 跨世界传送可能有 3 秒等待时间 |
| `/tpa <玩家ID>` | 请求传送到该玩家身边 | 需对方接受 `/tpaccept` |
| `/tpahere <玩家ID>` | 请求玩家传送到您身边 | 需对方接受 `/tpaccept` |
| `/msg <玩家> <内容>` | 发送**私密简讯** | 只有双方能看到的对话 |
| `/pwarp` | 访问**玩家地标** | 去参观大佬的刷怪塔或公共农场 |

## 🏆 技能与成就 (AuraSkills)
**查看您的等级和获取的荣誉。**

| 指令 | 功能说明 | 备注 |
| :--- | :--- | :--- |
| `/skills` | 查看**技能等级**菜单 | 包含采矿、农业、战斗等进度 |
| `/stats` | 查看您的**属性面板** | 力量、生命值、幸运值等数据 |
| `/ach` | 查看已获得的**成就** | 部分成就达成后有金币奖励 |
| `/title` | 管理并更换您的**称号** | 炫耀您的专属名号 |

## 🛡️ 安全与自检 (CoreProtect)
**如果您怀疑建筑被破坏或物品被盗，可以自行取证。**

| 指令 | 功能说明 | 备注 |
| :--- | :--- | :--- |
| `/co i` | 开启/关闭**交互查询** | 开启后点击方块查看破坏/放置记录 |
| `/pvp` | 切换 **PVP 状态** | 关闭后可防止被误伤 |
| `/suicide` | 选择**自杀** | 仅用于卡在无法脱出的地形时 |

## 💡 快捷键补充
* **按 `V` 键**：打开语音 Mod 菜单（需安装语音 Mod）。
* **按 `M` 键**：一键静音麦克风。
* **按 `Tab` 键**：查看当前在线玩家列表及其延迟 (Ping)。
