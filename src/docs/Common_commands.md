| **类别**        | **指令**                            | **功能说明**                 | **备注**               |
| --------------- | ----------------------------------- | ---------------------------- | ---------------------- |
| **AuthMe 验证** | `/register <密码> <密码>`           | 首次进入服务器时，注册账号。 | **必用** (离线模式)    |
|                 | `/login <密码>`                     | 再次进入服务器时，登录账号。 | **必用** (离线模式)    |
|                 | `/changepassword <旧密码> <新密码>` | 修改您的账号密码。           | 保护账号安全           |
| **基础操作**    | `/spawn`                            | 传送回服务器的默认出生点。   |                        |
|                 | `/sethome <名称>`                   | 设置一个家的位置。           | 数量限制：5            |
|                 | `/home <名称>`                      | 传送回您设置的家。           |                        |
|                 | `/tpa <玩家ID>`                     | 请求传送到另一位玩家身边。   | 需对方接受 `/tpaccept` |
|                 | `/tpahere <玩家ID>`                 | 请求另一位玩家传送到您身边。 | 需对方接受 `/tpaccept` |
|                 | `/msg <玩家ID> <信息>`              | 私聊发送信息给指定玩家。     | 常用指令               |
|                 | `/pvp`                              | 设定自己的pvp模式            |                        |
