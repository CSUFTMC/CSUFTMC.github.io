# 🏡 领地 (Residence) 玩家指南

欢迎！本服务器已开启**免费领地**功能，你可以通过以下简单的步骤保护你的建筑和财产。

---

## 🌍 允许圈地的世界
领地插件仅在以下世界可用，请务必在这些世界内安家：
* **主生存世界** (`world` / `main`)
* **绿洲世界** (`OasisWorld`)
* **地块城市** (`PlotsCity`)
> *注：资源大陆、地狱、末地及大厅已禁止圈地。*

---

## 🛠️ 核心工具
* **选择工具**：手持 **木锄 (Wooden Hoe)** 进行选区
* **查询工具**：手持 **线 (String)** 点击方块，可查看地块信息及权限。

---

## 📏 如何创建领地？
1.  **选取范围**：手持**木锄**，左键点击方块设置点 A，右键点击对角方块设置点 B。
2.  **全高度保护**（推荐）：输入 `/res select vert`。这会自动将你的选区扩展为从地底直达天空。
3.  **确认并命名**：输入以下指令完成创建：
    > `/res create [领地名字]`
    > *例如：/res create MyHome*

---

## 💰 玩家配额与限制 (免费)
为了保证服务器资源的合理分配，每位玩家享有以下免费额度：
* **领地数量**：每人最多可拥有 **3 个** 领地。
* **最大面积**：单个领地最大为 **64x64** (水平面积)。
* **高度区间**：支持从 **Y=-64** 到 **Y=320** 的全高度范围。
* **费用**：**完全免费 (0金币)**。
* **子领地**：每个领地内可划分 **3 个** 子区域 (Subzone)。

---

## 🔐 常用权限管理 (Flags)
想让小伙伴加入？站在领地内使用以下指令：

| 指令 | 用途 |
| :--- | :--- |
| `/res set` | 打开图形化管理界面 (最直观) |
| `/res pset [玩家] trusted t` | 给某人领地所有权限 (信任) |
| `/res pset [玩家] container t` | 仅给某人开箱权限 |
| `/res tp [领地名]` | 快速传送到领地 (需等待3秒) |
| `/res tpset` | 设置领地的传送落脚点 |
| `/res unstuck` | 如果你卡在领地里，用这个脱困 |

---

## ⚠️ 注意事项
1.  **自动清理**：如果连续 **60天** 未上线，系统将自动清除你的领地。
2.  **禁止区域**：地狱、末地及主城大厅禁止玩家创建领地。
3.  **删除领地**：输入 `/res remove [领地名]`。请谨慎操作，删除后无法撤回。
