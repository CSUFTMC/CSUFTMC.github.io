# 💰 商店教程

本服务器提供两种交易方式：**实体箱子商店**（QuickShop）和**全球虚拟市场**（GlobalShop）。

------

## 📦 QuickShop - 箱子商店

**适用场景**：在主世界、大厅建立自己的实体店铺，面对面交易。

### 1. 我要开店（出售物品）

1. 放置一个**箱子**。
2. **手持**你想要出售的物品。
3. **左键点击**箱子。
4. 在聊天栏输入**单价**（例如输入 `10`，代表 10金币/个）。
5. ✅ 成功！箱子上将会显示悬浮物品。

### 2. 我要收购（收购物品）

如果你想从其他玩家手中收购材料：

1. 按上述步骤建立一个普通的出售商店。
2. 对着商店输入指令：`/qs buy`。
3. 商店将切换为**收购模式**（需要确保箱子有空位，且你身上有足够的钱）。

### 3. 如何购买/出售

- **查看详情**：**左键**点击悬浮物品（查看库存、附魔、价格）。
- **进行交易**：**右键**点击箱子。
  - 如果要在聊天栏输入购买数量，请按提示操作（通常输入 `all` 可买光）。

### 4. 店主管理指令

对着自己的商店输入：

- `/qs price <新价格>`：修改价格。
- `/qs remove`：移除商店（变回普通箱子）。

------

## 🌏 GlobalShop - 全球市场

**适用场景**：随时随地买卖，无需跑图，类似“拍卖行”或“淘宝”。

### 1. 浏览市场

- 输入指令：`/gs`
- 点击界面中的图标即可购买其他玩家上架的物品。

### 2. 上架出售

1. **手持**你要卖的物品（如果是多个，请拿在手里）。
2. 输入指令：`/gs sell <总价格>`
   - *示例*：手拿 64 个钻石，输入 `/gs sell 1000`，代表这组钻石一共卖 1000 金币。
3. 物品将立即从背包扣除并上架。

### 3. 取回收益与物品

**重要**：如果东西卖出去了，或者过期没人买，物品/金币可能会进入“暂存箱”。

1. 输入 `/gs` 打开菜单。
2. 寻找 **“邮箱”** 或 **“暂存物品”** 图标（通常在界面底部）。
3. 点击领取你的金币或退回的物品。

------

## 💡 小贴士

- **QuickShop** 只能在你有权限的地方（如你的领地）创建。
- **GlobalShop** 上架可能会收取少量的手续费（视服务器设置而定）。
- 请勿上架违禁品，否则会被管理员没收！
