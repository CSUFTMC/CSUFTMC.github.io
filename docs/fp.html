假人插件使用指南

NickAe2026/2/13大约 6 分钟教程Minecraft插件

🤖 FakePlayer 假人插件使用指南

FakePlayer 是一款功能强大的服务器端插件，灵感来源于 Carpet-Mod。它允许你在 Minecraft 1.20.x 及更高版本服务器中生成“假人”。这些假人不仅看起来和真实玩家一样，还能保持区块加载，模拟玩家操作，是挂机刷怪、自动化生产的得力助手。

✨ 核心功能

  • 区块常加载：假人所在区块不会卸载，确保机器和农场 24 小时运行。
  • 真实玩家模拟：假人能像真实玩家一样被管理，支持各种原版和插件指令（如 /ban, /tp）。
  • 独立物品栏：可通过 /fp invsee 或右键点击假人来查看和编辑其物品栏。
  • 全面操作控制：完全控制假人的移动、跳跃、攻击等行为，甚至可以设置周期性动作。
  • 个性化配置：每个假人都可以有独立的配置，实现更灵活的用途。

🚀 快速上手

1. 召唤与移除假人

  • 召唤假人：在你当前位置生成一个假人。
    /fp spawn <名字>
# 示例: /fp spawn Bot_01
  • 移除假人：
    # 移除指定假人
/fp kill <名字>

# 移除所有假人 (需要OP权限)
/fp killall

2. 查看假人列表

如果不记得假人的名字或位置，可以查看在线假人列表：

/fp list

🎒 物品栏管理

你可以像管理自己的背包一样管理假人的物品（例如给它穿装备、给它工具挖掘）。

  • 指令方式：/fp invsee <名字>
  • 快捷方式：直接对着假人 右键点击 即可打开它的背包。
  • 丢弃物品：
    • /fp drop：丢弃手中物品。
    • /fp dropstack：丢弃手中整组物品。
    • /fp dropinv：清空假人背包内所有物品。
  • 交换主副手：/fp swap

🕹️ 动作与移动

假人可以执行各种周期性动作，非常适合用于刷怪塔或自动农场。

注意

若服务器上有多个假人，执行动作指令前，请先用 /fp select <名字> 选中目标，或看着指定的假人执行指令。

  • 持续攻击：/fp attack
    • 假人会持续挥动主手攻击前方。再次输入可停止。常用于刷怪塔。
  • 持续挖掘：/fp mine
    • 假人会持续挖掘面前的方块。再次输入可停止。
  • 持续交互：/fp use
    • 模拟持续按住右键，可用于放置方块、吃东西、拉拉杆等。再次输入可停止。
  • 停止所有动作：/fp stop
    • 让假人立即停止当前所有动作。
  • 移动与姿态：
    • /fp jump：让假人跳一下。
    • /fp sneak：切换潜行（下蹲）状态。
    • /fp sprint：切换疾跑状态。
    • /fp look：让假人看向你当前所指的位置。
    • /fp turn：让假人转身到你所看的方向。
    • /fp move：让假人移动。
    • /fp ride：让假人骑乘。
    • /fp tp：传送到假人身边。
    • /fp tphere：将假人传送到你身边。
    • /fp tps：与假人交换位置。

⚙️ 进阶配置

每个玩家都可以设置自己的假人配置，这些配置会在下次召唤假人时生效。

1. 修改假人默认行为

使用 /fp config set <选项> <true/false> 可以修改假人的默认行为配置。

  • 查看所有配置：/fp config list
  • 修改单个假人配置：/fp set <选项> <值> (针对已召唤的假人)
  • 修改默认配置：/fp config set <选项> <值> (会影响新召唤的假人)

2. 吸引怪物仇恨 (挂机必备) 🧟

默认情况下假人是无敌且不吸引仇恨的。如果你需要假人挂机刷怪塔（作为仇恨目标），必须关闭无敌模式。

/fp config set invulnerable false
# 设置为 false：关闭无敌（怪物会攻击假人，假人会掉血/饥饿）

⚠️ 重要：关闭无敌后，假人会死！请确保给假人提供信标回血或穿戴好防御装备。

3. 自动拾取物品 🧺

/fp config set pickup_items true
# 开启后，假人会自动捡起脚边的掉落物

4. 启用碰撞体积 🧱

/fp config set collidable true
# 开启后，假人会有碰撞箱（可以被推挤）

5. 自动换皮肤

/fp skin <正版玩家ID>
# 从指定玩家复制皮肤。注意：如果从离线玩家复制，可能有60秒冷却时间。

❓ 常见问题 (FAQ)

Q: 假人死亡后被踢出？

原因：一些插件（如 AuthMe）或服务器配置可能会将假人的断开连接视为异常。
解决：

  1. 修改服务器 config.yml 文件，将 prevent-kicking 设置为 ALWAYS。
    # config.yml
prevent-kicking: ALWAYS
  2. 若因未登录被踢出，可在假人的 self-commands 配置中添加登录指令。
    # 注意：请使用复杂密码，否则 AuthMe 可能拒绝
self-commands:
  - '/register <你的密码> <你的密码>'
  - '/login <你的密码>'
Q: 假人无法吸引怪物仇恨？

原因：默认情况下假人处于无敌模式 (invulnerable: true)，怪物会将其视为透明。
解决：

  1. 输入 /fp config set invulnerable false 关闭无敌模式。
  2. 关闭无敌后假人会受到伤害和饥饿，请确保提供回血或防御装备（如信标、金苹果）。
Q: 假人过一段时间自动退出？

原因：这可能是因为 AuthMe 等插件检测到假人长时间未登录或处于闲置状态。
解决：在假人的 self-commands 配置中添加自动登录指令（同上，确保设置密码）。
